-   tokenizer -  lex (flex 2.6.4) 
-   parser - yacc (bison (GNU Bison) 3.0.4) - 
-   Usually come pre-installed in linux.
-   If not, install with
    -   `sudo apt install flex`
    -   `sudo apt install bison`
